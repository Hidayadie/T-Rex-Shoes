## Fitur ##
**User**:
  - **Registrasi & Login**: user dapat membuat akun baru dan masuk dengan akun yang sudah ada.
  - **Katalog**: User dapat melihat daftar sepatu yang tersedia untuk dijual.
  - **Pembelian**: User dapat membeli sepatu yang ada di katalog yang nantinya akan masuk ke keranjang.
  - **Pencarian**: User bisa mencari sepatu berdasarkan nama atau merek.
  - **Keranjang**: User memiliki keranjang dan dapat melakukan checkout.

**Admin**:
  - **Manajemen Database Sepatu**: Menambahkan, mengedit, dan menghapus sepatu dari katalog.
  - **Manajemen Pengguna**: Melihat daftar pengguna, mengedit informasi pengguna, dan mengatur pengguna sebagai admin/operator.

## Instalasi & Running ##
**PENTING: IDE menggunakan codeblocks dan kompiler gcc**
<br>
belum dibuat cmake nya sih hehe, kloning repo ajah atau donlot versi yg udah rilis
1. Pada bagian kanan halaman repositori, terdapat keterangan **Releases** (Dibawah About)
2. Terdapat versi yang sudah tersedia disitu, Anda juga bisa klik **Releases** untuk lebih lengkap
3. Pilih versi lalu scroll ke bagian bawah, terdapat menu/pilihan assets dan 2 source code
4. Donlot versi zip jika anda berada di windows, sedangkan untuk linux gunakan tar.gz
5. Ekstrak file tersebut lalu jalankan kode berektensi cbp dengan `open project` menggunakan **CODE::BLOCKS**
6. Kalo bingung tanya discord: hytrin
